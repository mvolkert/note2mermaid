## ADR Format

Jede ADR folgt diesem Format:

- **Status:** Akzeptiert / Abgelehnt / Ersetzt durch ADR-XXX
- **Kontext:** Warum wurde eine Entscheidung benötigt?
- **Entscheidung:** Was wurde entschieden?
- **Begründung:** Warum diese Entscheidung?
- **Konsequenzen:** Was folgt daraus?
