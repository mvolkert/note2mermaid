## ADR Format

Each ADR follows this format:

- **Status:** Accepted / Rejected / Superseded by ADR-XXX
- **Context:** Why was a decision needed?
- **Decision:** What was decided?
- **Rationale:** Why this decision?
- **Consequences:** What follows from this?
